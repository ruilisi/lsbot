One Command to Start

Run lingti-bot relay --provider deepseek --api-key sk-xxx and get your personal bot page at https://bot.lingti.com/bots/xxx instantly. No platform account, no server, no domain needed.

19 Chat Platforms

Connect to WeCom, Feishu, DingTalk, WeChat, Slack, Telegram, Discord, WhatsApp, LINE, Teams, and more via cloud relay or self-hosted gateway.

16 AI Backends

Switch between Claude, DeepSeek, Kimi, MiniMax, Gemini, OpenAI, and 10 more providers per conversation. Supports Claude Extended Thinking out of the box.

75+ MCP Tools

Single 30MB binary. No Node.js, no Python, no Docker required to run. Works on macOS, Linux, Windows, ARM, MIPS — scp and run anywhere.

No client app needed. Open the browser, get multiple parallel sessions each with isolated AI memory. All sessions persist across page refreshes.
